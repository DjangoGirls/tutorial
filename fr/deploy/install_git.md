
### Windows

Vous pouvez télécharger Git depuis [git-scm.com][1]. Vous pouvez cliquer sur "suivant" (**next**) à toutes les étapes sauf une ; à la cinquième étape appelée "Ajustement de votre environnement PATH" (**Adjusting your PATH environment**), choisissez "Utiliser Git et les outils Unix optionels depuis l'Invite de commande Windows" (**Use Git and optional Unix tools from the Windows Command Prompt**, la dernière option). Autrement, les paramètres par défaut suffisent. "Checkout Windows-style, commit Unix-style line endings" est bien.

 [1]: http://git-scm.com/

### MacOS

Téléchargez Git depuis [git-scm.com][1] et suivez les instructions.

 [1]: http://git-scm.com/

### Linux

S'il n'est pas déjà installé, Git devrait être disponible via votre gestionnaire de paquets, donc essayez :


#### Debian ou Ubuntu

    $ sudo apt-get install git


#### Fedora (jusqu'à 21)

    $ sudo yum install git


#### Fedora (22+)

    $ sudo dnf install git
