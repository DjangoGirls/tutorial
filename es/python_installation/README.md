# Vamos a empezar con Python

¡Por fin estamos aquí!

Pero primero, déjanos decirte qué es Python. Python es un lenguaje de programación muy popular que puede ser usado para crear sitios web, juegos, software científico, gráficos, y más, mucho más.

Python se originó en la década de 1980 y su principal objetivo es ser legible por los seres humanos (¡no sólo por máquinas!). Por esta razón se ve mucho más simple que otros lenguajes de programación. This makes it easy to learn, but don't worry – Python is also really powerful!
